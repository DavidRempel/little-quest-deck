Little Quest Deck

A tiny deterministic card roguelite. Pick a monster, build a clean combo, then Attack. Random cards are weak. The whole selected set must match a pattern: all same number, all in sequence, or all same colour.

Rules

  • Each selected card is only worth 1 base point.
  • Same number: every selected card must have the same number.
  • Sequence: every selected card must form one unbroken run.
  • Same colour: every selected card must share a colour.
  • Set: number, colour, and symbol are each all same or all different.
  • Extra off-pattern cards break the combo. Do not spam the whole hand.
  • Discard selected cards once per monster for free; extra discards cost 1 HP.
