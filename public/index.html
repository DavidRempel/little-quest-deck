Little Quest Deck

A stripped-back card-combo prototype. Fight one enemy at a time. Each turn, choose one of three actions: Attack with selected cards, Draw One to grow your hand, or Discard X → Draw X to cycle cards without increasing hand size. Draw and cycle both let the enemy hit you for HP.

Rules

  • Attack with selected cards. The game automatically chooses the highest-scoring valid combo: Match, Run, Colour, or Set.
  • Score math: combo base + card count bonus + card value bonus. Card value bonus = ceil(card number / 3) for each played card.
  • Combo values: Match = 5 + 4/card; Run = 7 + 5/card; Colour = 4 + 3/card; Set = 9 + 6/card. Then add card value bonus.
  • Valid combos: Match = 2+ same number; Run = 3+ consecutive numbers; Colour = 2+ same colour; Set = 3+ numbers all same/all different and colours all same/all different.
  • Draw One adds one adventure card, up to a max hand size of 9, then the enemy attacks.
  • Discard X → Draw X discards as many selected cards as you want, then draws the same number back, so your hand does not grow.
  • Enemy attacks deal HP damage only in this build.
  • No treasures, rewards, relics, traits, enemy modifiers, or attack-type buttons in this build.
